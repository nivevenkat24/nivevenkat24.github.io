### **Overview**

A fully automated email intelligence agent that reads the top 50 emails each week, extracts themes, action items, and reminders, and sends a structured summary to the user.

---

### **Problem**

Professionals spend hours triaging inboxes, often missing critical to-dos hidden inside long threads. A lightweight, automated summarizer can dramatically reduce cognitive load and improve task prioritization.---

### **Architecture Diagram (High-Level RAG Pipeline)**

```mermaid
flowchart LR
  Cron[Weekly Cron Trigger] --> Gmail["Fetch 50 Emails"]
  Gmail --> LLM["OpenAI Summarizer"]
  LLM --> Organizer["Categorize: Highlights / To-Dos / Reminders"]
  Organizer --> EmailNode[Email Sender]
  EmailNode --> User
```
---
### **Tech Stack**
- n8n
- Gmail API
- GPT models
- Cron schedules
- Email node (SMTP)

---

### **Key Challenges**
- Avoiding invented tasks ("hallucinated action items")
- Handling duplicate content in long threads
- Designing summaries that work on mobile
- Ensuring reliable weekly execution

---

### **What I Learned**
- How to design multi-step agentic workflows
- How prompt engineering affects action-item extraction
- How to balance completeness and brevity
- How to deliver recurring automation with reliability
